## 📂 Exercises Breakdown

### 🔹 Exercise 1 — OOP & Core Logic Systems
> Built reusable class-based systems in Python, demonstrating object-oriented principles including classes, objects, inheritance, encapsulation, and method overriding.

**Key Concepts:** `Classes` · `Objects` · `Inheritance` · `Encapsulation` · `Abstraction`

---

### 🔹 Exercise 2 — Data Preprocessing Pipeline
> Developed a complete data preprocessing pipeline to prepare raw datasets for machine learning, covering missing value handling, outlier detection, categorical encoding, and feature scaling.

**Key Concepts:** `Pandas` · `NumPy` · `Missing Values` · `Label Encoding` · `Feature Scaling`

---

### 🔹 Exercise 3 — Linear Regression Model
> Trained and evaluated a Linear Regression model from scratch on a real dataset, with visualizations of model fit, residuals, and performance metrics.

**Key Concepts:** `Scikit-Learn` · `Linear Regression` · `R²` · `MSE` · `MAE` · `Matplotlib`

---

### 🔹 Retail Sales Analysis — EDA Capstone Project
> End-to-end Exploratory Data Analysis on a 1,825-row retail sales dataset. Covers full preprocessing, trend analysis, category & regional performance, day-of-week patterns, correlation heatmap, and a 4-chart business dashboard with actionable insights.

**Dataset:** 1,825 rows · 6 columns (Date, Category, Sales, Quantity, Profit, Region)

**Key Concepts:** `EDA` · `Time Series` · `Data Cleaning` · `Pandas` · `Matplotlib` · `Seaborn` · `Business Insights`

| Analysis | What Was Done |
|----------|--------------|
| 📅 Monthly Trend | Sales & profit tracked across all 12 months |
| 🏷️ Category Performance | Revenue & margin breakdown by product category |
| 🗺️ Regional Analysis | Sales distribution across North, South, East, West |
| 📆 Day-of-Week Pattern | Identified peak and low sales days |
| 🔥 Correlation Heatmap | Relationships between Sales, Profit, Quantity, Month |
| 📊 4-Chart Dashboard | Combined business summary visualization |

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/daniyal-devx/DigiSkills-AI-Using-Python.git

# 2. Navigate into the project
cd DigiSkills-AI-Using-Python

# 3. Install dependencies
pip install numpy pandas matplotlib seaborn scikit-learn jupyter

# 4. Open any exercise or project
cd Retail-Sales-Analysis
jupyter notebook
```
