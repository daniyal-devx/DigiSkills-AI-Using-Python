## 📌 About This Repository

This repository contains all my **Hands-on Exercise submissions** for the **DigiSkills AI Using Python** course — one of Pakistan's largest free digital skills training programs. Each folder represents a graded exercise covering a specific AI/ML concept, implemented in Python.

## 📂 Exercises Breakdown

### 🔹 Hands-on Exercise 1 — OOP & Core Logic Systems
> Built reusable class-based systems in Python, demonstrating object-oriented principles including classes, objects, inheritance, encapsulation, and method overriding.

**Key Concepts:** `Classes` · `Objects` · `Inheritance` · `Encapsulation` · `Abstraction`

---

### 🔹 Hands-on Exercise 2 — Data Preprocessing Pipeline
> Developed a complete data preprocessing pipeline to prepare raw datasets for machine learning, covering missing value handling, outlier detection, categorical encoding, and feature scaling.

**Key Concepts:** `Pandas` · `NumPy` · `Missing Values` · `Label Encoding` · `Feature Scaling`

---

### 🔹 Hands-on Exercise 3 — Linear Regression Model
> Trained and evaluated a Linear Regression model from scratch on a real dataset, with visualizations of model fit, residuals, and performance metrics.

**Key Concepts:** `Scikit-Learn` · `Linear Regression` · `R²` · `MSE` · `MAE` · `Matplotlib`

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/digiskills-ai-python.git

# 2. Navigate into the project
cd digiskills-ai-python

# 3. Install dependencies
pip install numpy pandas matplotlib scikit-learn jupyter

# 4. Open any exercise
cd Hands-on-Exercise-1
jupyter notebook
```

> 💡 You can also open any `.ipynb` file directly in [Google Colab](https://colab.research.google.com/) — no setup needed.
